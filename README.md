# MindKey — Static Site (Free + Premium quiz)

This repository contains a static site prototype for MindKey:
- `index.html`, `how.html`, `about.html`
- `quiz_free.html` (10 questions)
- `quiz_premium.html` (30 questions)
- `styles.css` + `app.js`

## How to deploy

### Option A — GitHub Pages (quick)
1. Create a GitHub repo (e.g., `mindkey`).
2. Upload the `public/` contents to the repository root (or put them in `gh-pages` branch).
3. In repo Settings → Pages choose branch `main` and folder `/root` (or `gh-pages`) and save.
4. After a minute, your site will be at `https://<your-username>.github.io/<repo-name>/`.

### Option B — Netlify
1. Create account on Netlify.
2. Drag & drop the folder (with the files) into Netlify deploy area.
3. Site is published instantly.

### Option C — Vercel
1. Create account on Vercel and connect GitHub repo.
2. Select repo and deploy (static).
